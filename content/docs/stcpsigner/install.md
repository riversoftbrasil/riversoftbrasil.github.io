1. Insira o CD na unidade de leitura.

2. Em **Iniciar**, escolha a opção **Executar**.

3. Utilize o botão **Procurar** para acessar a unidade de leitura de CD.

4. Selecione o programa **SETUP.EXE**.

5. Execute o programa.
